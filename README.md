## Installation

Install git-portfolio with yarn

```bash
  yarn install
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/djarwo/basic-material-portfolio
```

Go to the project directory

```bash
  cd djarwoprasojo-portfolio
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
